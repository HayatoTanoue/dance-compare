# Dance Compare - ダンス練習アプリ

ダンスの練習が一人でできるようなインターフェースと解析機能を持ったアプリケーション

## 🎯 主な機能

### 📹 動画処理機能
- **教師動画・生徒動画のアップロード**: ドラッグ&ドロップまたはファイル選択
- **MediaPipe骨格推定**: リアルタイム骨格ランドマーク検出
- **骨格重畳表示**: 骨格ランドマークを動画上に可視化
- **ブラウザ互換動画生成**: FFmpegによるH.264変換で確実な再生

### 📊 分析機能
- **カラオケ風採点システム**: 一定間隔での関節動きの一致度評価
- **時間同期機能**: 教師・生徒動画の再生開始時刻調整
- **関節レベル分析**: 全体・上半身・下半身・体幹・腕・脚別の詳細分析
- **リアルタイム比較**: 類似度スコアの時系列グラフ表示

## 🛠️ 技術仕様

- **フロントエンド**: Streamlit
- **バックエンド**: Python 3.12+
- **骨格推定**: MediaPipe Pose
- **動画処理**: OpenCV + FFmpeg
- **可視化**: Plotly, Matplotlib

## 📋 セットアップ手順

### 1. 依存関係のインストール

```bash
# リポジトリをクローン
git clone https://github.com/HayatoTanoue/dance-compare.git
cd dance-compare

# 仮想環境の作成（推奨）
python -m venv venv
source venv/bin/activate  # Linux/Mac
# または
venv\Scripts\activate  # Windows

# 依存関係のインストール
pip install -r requirements.txt
```

### 2. FFmpegのインストール

```bash
# Ubuntu/Debian
sudo apt update && sudo apt install ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows
# https://ffmpeg.org/download.html からダウンロード
```

### 3. アプリケーションの起動

```bash
streamlit run app.py
```

ブラウザで `http://localhost:8501` にアクセス

## 📖 使用方法

### 基本的なワークフロー

1. **動画アップロード**
   - 左側: 教師動画をアップロード
   - 右側: 生徒動画をアップロード
   - 対応形式: MP4, AVI, MOV

2. **骨格推定の実行**
   - 各動画で「骨格推定を実行」ボタンをクリック
   - MediaPipeが自動的に骨格ランドマークを検出

3. **骨格重畳表示**
   - 「骨格重畳表示」チェックボックスを有効化
   - 骨格ランドマークが重畳された動画が生成・表示

4. **比較分析**
   - 時間同期設定で開始時刻を調整
   - 分析レベル（全体・部位別）を選択
   - 「比較分析を実行」で詳細な類似度分析

### 分析機能の詳細

#### カラオケ風スコア
- 一定間隔（0.5-3秒）での類似度を色分け表示
- 🥇 金色: 80%以上（優秀）
- 🟢 緑色: 60-80%（良好）
- 🟠 オレンジ: 40-60%（普通）
- 🔴 赤色: 40%未満（要改善）

#### 分析レベル
- **全体**: 全33関節の総合評価
- **上半身**: 肩・腕・胴体の動き
- **下半身**: 腰・脚・足の動き
- **体幹**: 胴体の安定性
- **左腕/右腕**: 腕の個別動作
- **左脚/右脚**: 脚の個別動作

## 🧪 テスト手順

### 1. 基本機能テスト

```bash
# テスト用動画の生成
python create_test_video.py

# アプリケーションの起動
streamlit run app.py
```

### 2. 骨格推定テスト

1. ブラウザで `http://localhost:8501` にアクセス
2. `teacher_test.mp4` と `student_test.mp4` をアップロード
3. 両方の動画で骨格推定を実行
4. 「骨格推定完了！」メッセージを確認

### 3. 骨格重畳表示テスト

1. 各動画で「骨格重畳表示」チェックボックスを有効化
2. 骨格ランドマークが重畳された動画が表示されることを確認
3. 動画再生コントロール（再生/一時停止）が正常に動作することを確認
4. 骨格ランドマーク（白い線）が動画上に正確に描画されることを確認

### 4. 比較分析テスト

1. 両方の動画で骨格推定完了後、「比較分析を実行」をクリック
2. カラオケ風スコアグラフが表示されることを確認
3. 詳細分析の時系列グラフが表示されることを確認
4. 分析サマリーの統計情報が表示されることを確認

### 5. 動画互換性テスト

```bash
# 骨格重畳表示動画の内容確認
python test_overlay_video.py

# ブラウザ互換性テスト
python test_browser_compatible.py
```

## 🔧 トラブルシューティング

### よくある問題

#### 1. 骨格重畳表示動画が再生されない
- **原因**: FFmpegがインストールされていない
- **解決**: FFmpegをインストールしてアプリを再起動

#### 2. 骨格推定が失敗する
- **原因**: 動画形式が対応していない、または破損している
- **解決**: MP4形式の動画を使用、動画ファイルを確認

#### 3. 動画アップロードができない
- **原因**: ファイルサイズが大きすぎる
- **解決**: 動画を圧縮するか、短い動画を使用

#### 4. 分析結果が表示されない
- **原因**: 両方の動画で骨格推定が完了していない
- **解決**: 両方の動画で骨格推定を実行してから分析を実行
